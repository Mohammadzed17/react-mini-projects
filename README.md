# React Mini Projects

A collection of small React applications built while learning core React concepts such as hooks, state management, rendering lists, and component structure.

---

## 🚀 Projects Included

### 1️⃣ To-Do List
- Add tasks
- Delete tasks
- Move tasks up and down
- State management using useState
- Array updates using spread operator

### 2️⃣ Digital Clock
- Real-time clock
- useEffect with setInterval
- Cleanup function
- Date object formatting

### 3️⃣ Stopwatch
- Start, Stop, Reset functionality
- useRef for interval control
- useState for time tracking
- Proper cleanup handling

### 4️⃣ Color Picker
- Live color preview
- Controlled inputs
- useState hook
- Dynamic styling

---

## 🛠️ Built With

- React (Vite)
- JavaScript (ES6+)
- React Hooks:
  - useState
  - useEffect
  - useRef

---

## ▶️ How to Run Any Project

1. Navigate into a project folder:

```
cd project-folder-name
```

Example:

```
cd todo-list
```

2. Install dependencies:

```
npm install
```

3. Run development server:

```
npm run dev
```
